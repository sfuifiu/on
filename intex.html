About the Project

This project is a water quality monitoring system designed for aquaculture applications. It continuously tracks key parameters such as:

  • Turbidity – to detect water clarity
  • Temperature – to maintain optimal fish health
  • TDS (Total Dissolved Solids) – to measure water purity

The system uses sensors connected to a Raspberry Pi and sends data to ThingSpeak, where it can be viewed and exported. This real-time monitoring helps improve fish yield, reduce risk, and support sustainable aquaculture.
